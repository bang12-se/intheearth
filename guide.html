이용 가이드

처음 방문했다면 메인 페이지 상단 소개를 먼저 읽은 뒤, 행성별 설명 카드와 인터랙티브 보기 순서로 이동하는 것을 권장합니다. 이 사이트는 단일 기능 데모가 아니라, 읽는 콘텐츠와 시각적 탐색을 함께 제공하는 구조로 설계되었습니다.

추천 사용 순서

  1. 메인 페이지의 사이트 소개와 콘텐츠 구성 설명을 읽습니다.
  2. 지구, 화성, 목성처럼 차이가 큰 천체를 먼저 선택해 핵심 특징을 비교합니다.
  3. 인터랙티브 화면에서 회전과 확대를 사용해 시각적 차이를 확인합니다.
  4. 필요하면 운영 문서와 문의 페이지를 통해 사이트 정책을 확인합니다.

모바일 사용 팁

모바일 기기에서는 성능 안정을 위해 일부 렌더링 품질이 자동 조정될 수 있습니다. 그래도 행성 선택, 드래그, 확대·축소, 정보 패널 확인 같은 핵심 기능은 동일하게 사용할 수 있습니다.

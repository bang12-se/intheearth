      서비스 소개

      Geo Signal Tracker는 태양계 천체를 인터랙티브 3D 스타일 화면으로 탐색할 수 있는 소개형 시뮬레이션입니다. 드래그, 줌, 틸트 컨트롤로 시점을 바꾸며 각 행성의 분위기와 시각적 특징을 직관적으로 살펴볼 수 있습니다.

      지구의 특징

      • 표면의 약 71%가 바다로 덮여 있어 우주에서 푸른 행성으로 보입니다.
      • 질소와 산소 중심의 대기가 생명 유지와 기후 안정에 핵심 역할을 합니다.
      • 강한 자기장이 태양풍을 막아 지표 환경을 보호합니다.
      • 달의 중력 영향으로 조석 현상이 발생하고 자전축 안정성에 도움을 줍니다.

      콘텐츠 품질 원칙

      • 중복 페이지를 만들지 않고, 단일 사이트에서 독창적인 정보를 제공합니다.
      • 사용자가 원하는 정보를 빠르게 찾도록 명확한 탐색 구조를 유지합니다.
      • 광고보다 콘텐츠 가치를 우선하며, 과도한 광고 배치를 지양합니다.
      • 오해를 유도하는 문구나 클릭 유도형 인터페이스를 사용하지 않습니다.

      이용 가이드

      1. 상단 행성 선택 메뉴에서 보고 싶은 천체를 선택합니다.
      2. 마우스/터치 드래그로 시점을 회전하며 표면과 조명 변화를 확인합니다.
      3. `+ / - / Tilt / Compass`로 확대, 축소, 기울기, 방향을 조절합니다.
      4. 하단 상태바에서 현재 방향, 고도, 타겟 행성 정보를 확인합니다.
